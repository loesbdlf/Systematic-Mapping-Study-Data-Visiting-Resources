# Systematic-Mapping-Study-Data-Visiting-Resources

This repository contains the the resources used during a Systematic Mapping Study on Data Visiting architectures, for the paper ```Architectural Patterns for Bringing Computation to Data: A Systematic Mapping Study on the Data Visiting Paradigm.```

The repository is set up as follows.

- Data extraction /
    - 7D-DVF adaption
    - Data extraction table
    - Extracted data
    - Architecture graphs /
        - Graph per architecture / pattern
- Data analysis /
    - Statistical analysis scripts /
        - Script for multivariate analysis
        - Script for univariate analysis
    - Statistical results
